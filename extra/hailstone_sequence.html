Hailstone σειρά

Η σειρά hailstone παράγεται ως εξής:

Ξεκινάει με έναν αριθμό έστω n, αν ο αριθμός είναι ζυγός τότε n = n/2 διαφορετικά n = 3n+1.

Η διαδικασία συνεχίζεται μέχρι να καταλήξει στον αριθμό 1.

Παραδείγματα
2, 1
3, 10, 5, 16, 8, 4, 2, 1
4, 2, 1
2n, 2n-1 , ... , 4, 2, 1
5, 16, 8, 4, 2, 1
7, 22, 11, 34, 17, 52, 6, 13, 40, ...?
Παράδειγμα

Στο παρακάτω διάγραμμα ροής περιγράφεται ο αλγόριθμος της σειράς

// Java
int n = 3;
while (n != 1) {
	System.out.println(n);
	if (n % 2 == 0) {
		n = n / 2;
	} else {
		n = 3 * n + 1;
	}
}
System.out.println(n);
Παράδειγμα

Στο παρακάτω διάγραμμα ροής περιγράφεται ο αλγόριθμος της σειράς όπως αυτή παράγεται μετά από εισαγωγή αρχικής τιμής από χρήστη

// Java
int n;
n = Integer.parseInt(JOptionPane.showInputDialog("δώσε αριθμό"));
while (n != 1) {
	System.out.println(n);
	if (n % 2 == 0) {
		n = n / 2;
	} else {
		n = 3 * n + 1;
	}
}
System.out.println(n);
Παράδειγμα

Ο αλγόριθμος με τη χρήση λίστας (List)

ArrayList list = new ArrayList();
int n = 3;
while (n != 1) {
	list.add(n);
	if (n % 2 == 0) {
		n = n / 2;
	} else {
		n = 3 * n + 1;
	}
}
list.add(n);

//Iterating the list
Iterator iterator = list.iterator();
while(iterator.hasNext()) {
	System.out.println(iterator.next());
}
